## Dokumentasi


# Brainstorm — Muhammad Rozinul Mahrus

## Prompt 1
**Konteks:**
Role: UX/UI consultant untuk portfolio developer.
Task: Bantu brainstorm desain portfolio untuk fresh graduate IT.
Context: Muhammad Rozinul Mahrus (React, Node.js, Tailwind, MySQL) - Target Startup/Corporate.
Constraints: Mobile-first, Simple tapi stand out.
Output: 3 color options, Typography, Section structure, 1 unique element.

## AI Response 1
Menyediakan 3 opsi warna:
1. Modern Tech (Blue/Slate) - Aman & Profesional untuk Corporate
2. Dark Mode Minimalist (Dark/Emerald) - Fokus & Developer-feel untuk Startup
3. Soft Neo-Brutalism (Cream/Orange) - Trendy & Berani

Rekomendasi Font: Plus Jakarta Sans / Outfit (Heading) & Inter (Body).
Struktur (Single Page): Hero -> Tech Stack Grid -> 3 Featured Projects -> About & Education -> Contact.
Unique Element: "Profile/Code Toggle Card" (Kartu profil yang bisa diflip menjadi block code syntax React).

## Prompt 2
Bahas semuanya satu per satu secara mendalam:
1. Kenapa pilih warna ini?
2. Inter vs Poppins, mana lebih cocok?
3. Single page vs multi-page?
4. Bagaimana beda dari candidate lain?

## AI Response 2
1. **Warna:** Dark mode sangat melekat dengan habitat developer, memberi kesan "tech-savvy".
2. **Typography:** Inter lebih cocok dari Poppins karena lebih profesional, serius, dan rapi untuk UI digital (seperti SaaS), sementara Poppins kadang terlalu santai.
3. **Layout:** Single page jauh lebih baik karena HRD hanya punya waktu singkat. Konten fresh grad terasa lebih padat di 1 halaman & *scrollable* di mobile.
4. **Stand Out:** Fokus deskripsi project pada "Problem & Solution", jangan cuma sebut fitur. Buat elemen interaktif interaktif spesifik untuk developer seperti toggle "View as Code".


## Key Decisions
| Decision | Value | Why |
|----------|-------|-----|
| Color | Dark Mode Minimalist (Gelap + Cyan/Emerald) | Kesan "tech-savvy", disukai startup, & membuat warna thumbnail project menonjol. |
| Typography | Inter (Body) & Plus Jakarta Sans (Heading) | Inter didesain untuk UI digital sehingga rapi & profesional, Plus Jakarta Sans memberi kesan modern. |
| Layout | Single Page | Mudah dan cepat di-scan dalam 1-2 menit oleh HRD/Tech Lead, sangat ideal untuk *Mobile-first*. |
| Style | Deskripsi "Problem & Solution" + "View as Code" | Menunjukkan pola pikir problem solver (engineer) dan membuktikan kemampuan teknis (React/Tailwind) secara langsung. |
